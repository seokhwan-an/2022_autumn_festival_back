# 2022_autumn_festival_back good

## 🐘 2022 동국대학교 가을축제 사이트 (BE)

## 프로젝트 배경
축제와 관련된 정보들이 여러 SNS(인스타그램, 에브리타임 등)로 분산되어 학우들이 축제와 관련된 정보를 찾는데 불편함을 느껴 축제 서비스를 개발 했습니다.
